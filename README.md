## Parsemedia

App Inventor расширение для анализа медиафайлов — определения типа файла (фото или видео) и извлечения даты создания из метаданных.

### Возможности

1. **IsPhoto** — проверяет, является ли файл изображением (JPG, PNG, GIF, WebP, HEIC, RAW и др.)

![Компонент проверки на фото](\doc\isPhoto.png)

На вход:
- `filePath` — путь к файлу на устройстве (текстовая строка)

На выход:
- `true`, если файл является изображением; иначе `false`

***

2. **IsVideo** — проверяет, является ли файл видео (MP4, MOV, MKV, WebM, AVI и др.)

![Компонент проверки на видео](\doc\isVideo.png)

На вход:
- `filePath` — путь к файлу на устройстве (текстовая строка)

На выход:
- `true`, если файл является видео; иначе `false`

***

3. **GetDate** — возвращает дату создания в формате `yyyy:MM:dd HH:mm:ss`:
  - для фото — из EXIF-метаданных (`DateTimeOriginal`, `DateTime`, `DateTimeDigitized`);
  - для видео — через `MediaMetadataRetriever`.

![Компонент получения даты](\doc\getDate.png)

На вход:
- `filePath` — путь к файлу на устройстве (текстовая строка)

На выход:
- дата создания в формате `yyyy:MM:dd HH:mm:ss`
- `"No date found"`, если дата в метаданных отсутствует
- пустая строка `""`, если файл не удалось прочитать


### Сборка

Расширение создано с помощью [Rush](https://github.com/shreyashsaitwal/rush-cli).
